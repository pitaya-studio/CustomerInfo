子表格设置

容器(qc-sub-table-container)定义需要进行子表格的区域,通常设置在table或者table的父级容器上

切换按钮(qc-sub-table-anchor),通过点击来切换子表格的显示与否

子表(qc-sub-table)子表格其实是在主表格的一个tr上合并所有的单元格,然后将子表格填充到该单元格

子表id(qc-sub-table-id),设置相同的qc-sub-table-id 使切换按钮和子表完成交互

   <table class="table table-bordered" qc-sub-table-container>
        <thead>

            <th width="20%">价格</th>
            <th width="20%">操作</th>
        </tr>
        </thead>
        <tbody>

            <td>¥131,420.00</td>
            <td>
                <button qc-sub-table-anchor qc-sub-table-id="order1">toggle order</button>
                <button qc-sub-table-anchor qc-sub-table-id="sale1">toggle sale</button>
            </td>

调整列宽

准备工作

依赖项:angular.js,qc.basic.js,qc.js

依赖模块:qc module

qc-table-resizecolumn:只需要给表格添加class(qc-table-resizecolumn)即可进行列宽调整

示例

    <table class="table table-bordered qc-table-resizecolumn">
       ...
    </table>

表格操作项

准备工作

依赖项:angular.js,qc.basic.js,qc.js

依赖模块:qc module

qc-table-operator:只需要给表格总的操作项容器添加class(qc-table-operator)即可

示例

    <td>
        <div class="qc-table-operator">
            <span ng-repeat-start="o in item.operator" ng-if="!$first"> |</span>
            <a class="" ng-repeat-end="">{{o}}</a>
        </div>
    </td>
